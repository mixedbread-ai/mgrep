# Verify

Run full verification suite before committing changes.

## Steps

1. Run type checking:
   ```bash
   pnpm typecheck
   ```

2. Run linting:
   ```bash
   pnpm lint
   ```

3. Run tests:
   ```bash
   pnpm test
   ```

4. If any step fails, fix the issues before proceeding.

5. Report the results with a summary of any issues found.
